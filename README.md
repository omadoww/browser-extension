# Dependencies Linker for GitHub

The "Dependencies Linker for GitHub" is a Google Chrome Extension which links [npm](http://npmjs.org) and [bower](http://bower.io) dependencies listed in package.json / bower.json on GitHub to their projectâ€™s pages. It also links require() statements with the related file or package.
